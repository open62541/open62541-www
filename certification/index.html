OPC UA Certification

OPC Foundation Certified

Certified Implementation

The sample server built using open62541 has been certified by the OPC Foundation for compliance with the Standard Server 2017 profile.

This certification demonstrates that open62541 provides a robust, standards-compliant OPC UA implementation suitable for production use.

Certification Details

The OPC Foundation Certified for Compliance logo is a trademark of the OPC Foundation and may be used only by written permission. This certification indicates that our implementation has been tested by an independent certification lab and meets all requirements of the following specification:

Profile

Standard Server 2017

Security Policies

  • • Basic128Rsa15
  • • Basic256
  • • Basic256Sha256
  • • Aes128-Sha256-RsaOaep
  • • Aes256-Sha256-RsaPss

Key Features

  • • Client/Server communication
  • • Subscriptions & monitoring
  • • Method calls
  • • Node management
  • • Historical data access
  • • Events and alarms
  • • Security & authentication

About the Certification

The certified implementation is based on the open62541 SDK and demonstrates the full capabilities of the library. Organizations using open62541 can reference this certification when developing their own OPC UA applications.

Important Notes:

  • Certification applies to the reference server implementation
  • Applications can leverage the certified codebase
  • Contact OPC Foundation Test Lab for product-specific certification
